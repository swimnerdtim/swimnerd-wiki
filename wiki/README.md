# Swimnerd Swimming Knowledge Wiki

This wiki contains compiled swimming knowledge from championship coaches, training methodology books, and race analysis data.

## Structure

- `/coaches/` - Coach-specific knowledge (Bob Bowman, etc.)
- `/technique/` - Stroke technique and mechanics
- `/methodology/` - Training philosophy, periodization, tapering
- `/workouts/` - Practice templates and training sets

## Usage

This wiki serves as the knowledge base for Swimnerd AI. Pages are referenced dynamically based on user questions.

- Race split analysis
