# Schneider Workout Structure & Terminology

**Source:** *The Swimmer's Workout Handbook* by Terri Schneider  
**Context:** How to read, write, and structure swimming workouts using Schneider's system  
**Audience:** Coaches writing workouts, swimmers learning to interpret workout cards

---

## Overview: The Anatomy of a Swimming Workout

Schneider's handbook contains **100 progressive workouts** ranging from 1000-5000 yards/meters. Each follows a predictable structure that makes them easy to read, execute, and scale.

**Key insight:** Swimming workouts aren't continuous. They're composed of **sets** with built-in rest, organized by purpose (warm-up, drills, main set, cool-down).

> "20-30 minutes ≠ non-stop swimming (difficult to impossible for beginners). Reality: Swimmers take breaks. Seldom swim more than 400-500 yards at a time (unless distance training). Combination of shorter distances: 50s, 100s, 200s." — *Championship Swimming*

## Core Workout Terminology

### Basic Set Notation

#### 4×50
**Translation:** Swim 50 yards/meters, 4 times (total = 200)

**Full notation:**  
`4×50 freestyle (L3) w/:15R`

**Breakdown:**
- **4×** = number of repeats
- **50** = distance per repeat (yards or meters)
- **freestyle** = stroke
- **(L3)** = training level (high-end aerobic to low anaerobic)
- **w/:15R** = with 15 seconds rest between each 50

**How to execute:**
1. Swim 50 freestyle at L3 intensity
2. Rest 15 seconds
3. Repeat 3 more times (total = 4)

---

#### Interval vs. Rest Notation

**Two ways to write rest:**

**Method 1: Rest Time (w/:XXR)**
- `6×100 w/:20R` = 20 seconds rest after each 100
- Rest starts when you touch the wall

**Method 2: Interval Time (on :XX or on X:XX)**
- `10×50 on 1:15` = Start a new 50 every 1:15
- If you finish in :55, you get :20 rest
- If you finish in 1:05, you get :10 rest

**Schneider primarily uses rest notation (w/:XXR)**, but coaches should know both.

---

### Distance Progressions

#### 50, 100, 200, 100, 50
**Translation:** Pyramid set
1. Swim 50
2. Swim 100
3. Swim 200
4. Swim 100
5. Swim 50
**Total = 500**

**Full notation:**  
`50, 100, 200, 100, 50 freestyle (L4) w/:30R`

**Rest:** 30 seconds after each swim (not just at the end)

---

#### 2×50, 75, 100, 75, 2×50
**Translation:** Double-bookend pyramid
1. Swim 50, rest
2. Swim 50, rest
3. Swim 75, rest
4. Swim 100, rest
5. Swim 75, rest
6. Swim 50, rest
7. Swim 50
**Total = 500**

---

### Combo Swims (Multiple Strokes in One Interval)

#### 100; 50 butterfly/50 backstroke
**Translation:** One 100-yard/meter swim, switching strokes halfway

**How to execute:**
1. Swim 50 butterfly
2. At the 50 mark (wall or mid-pool), switch to backstroke
3. Swim 50 backstroke
4. **Total = 100** (counts as ONE interval)

**Why the semicolon?**
- Separates different parts of a single swim
- **Comma** = separate intervals with rest between
- **Semicolon** = single interval with internal structure

---

#### 150; 50 freestyle/50 breaststroke/50 freestyle
**Translation:** 150-yard/meter swim with stroke changes

1. Swim 50 freestyle
2. Swim 50 breaststroke
3. Swim 50 freestyle
**Total = 150** (one interval)

---

#### 4×50; 25 backstroke/25 breaststroke
**Translation:** Four 50s, each one split 25 back/25 breast

**How to execute:**
1. Swim 25 backstroke
2. Flip/turn, swim 25 breaststroke
3. Rest (per notation, e.g., :15R)
4. Repeat 3 more times
**Total = 200** (4 intervals of 50 each)

---

### Drill/Swim Combinations

#### 4×50; 25 catch up drill/25 freestyle w/:15R
**Translation:** Four 50s, first 25 is drill, second 25 is regular swim

**How to execute:**
1. Swim 25 catch-up drill
2. Swim 25 freestyle
3. Rest 15 seconds
4. Repeat 3 more times
**Total = 200**

---

#### 200 alternate; 25 fingertip drag drill/25 freestyle
**Translation:** 200-yard/meter continuous swim, alternating every 25

**How to execute:**
1. 25 fingertip drag drill
2. 25 freestyle
3. 25 fingertip drag drill
4. 25 freestyle
5. 25 fingertip drag drill
6. 25 freestyle
7. 25 fingertip drag drill
8. 25 freestyle
**Total = 200** (no rest — one continuous swim)

**Key word:** "alternate" = keep switching back and forth

---

### Set Modifiers

#### Descend the Set
**Translation:** Get faster with each interval

**Example:**  
`6×50 freestyle descend the set w/:15R`

**Execution:**
1. Rep 1: Moderate (e.g., :40)
2. Rep 2: Faster (:38)
3. Rep 3: Faster (:36)
4. Rep 4: Faster (:34)
5. Rep 5: Faster (:32)
6. Rep 6: Fastest (:30)

**Goal:** Each swim is faster than the previous one.

---

#### Build / Build Each
**Build:** Within one swim, start easy and finish fast  
**Build each:** Each swim in the set is faster than the last

**Example (Build):**  
`4×50 freestyle build w/:15R`
- Each 50: first 25 easy, second 25 fast

**Example (Build Each):**  
`2×100 freestyle build each w/:15R`
- Rep 1: 1:30
- Rep 2: 1:25 (faster than rep 1)

---

#### Fastest Average
**Translation:** Swim the best pace you can hold for ALL reps (not just one fast swim)

**Example:**  
`6×100 freestyle fastest average w/:40R`

**Goal:** All 6 intervals should be within 2-3 seconds of each other.

**NOT this:**
- Rep 1: 1:05 (too fast)
- Rep 6: 1:25 (died)

**THIS:**
- All reps: 1:18-1:20 (consistent, sustainable max pace)

---

#### Fastest Time
**Translation:** Your absolute best time for that distance (all-out effort)

**Example:**  
`100 freestyle fastest time`

**Execution:** Sprint, empty the tank, record your time.

---

### Training Level Modifiers

#### 1-4 (L3), 5-8 (L4)
**Translation:** In a set of 8, first 4 at Level 3, second 4 at Level 4

**Example:**  
`8×75 freestyle w/:15R; 1-4 (L3), 5-8 (L4)`

**Execution:**
- Reps 1-4: High-end aerobic (L3)
- Reps 5-8: Lactate threshold (L4)

---

#### Odds (L3), Evens (L5)
**Translation:** Odd-numbered reps at L3, even-numbered reps at L5

**Example:**  
`6×50 freestyle w/:20R; odds (L3), evens (L5)`

**Execution:**
- Rep 1: L3 (moderate)
- Rep 2: L5 (all-out)
- Rep 3: L3
- Rep 4: L5
- Rep 5: L3
- Rep 6: L5

**Purpose:** Contrast training — teaches body to recover while still swimming.

---

### Special Formats

#### Locomotion
**Translation:** Sprint/easy pattern that escalates in distance

**For 300:**
- 25 sprint / 25 easy
- 50 sprint / 50 easy
- 75 sprint / 75 easy
**Total = 300**

**For 500:**
- 25 sprint / 25 easy
- 50 sprint / 50 easy
- 75 sprint / 75 easy
- 100 sprint / 100 easy
**Total = 500**

**Example:**  
`300 pull locomotion`

**Execution:**
1. 25 pull (sprint), 25 pull (easy)
2. 50 pull (sprint), 50 pull (easy)
3. 75 pull (sprint), 75 pull (easy)

**What it teaches:** Speed changes, lactate tolerance, mental toughness

---

#### Broken At
**Translation:** Pause the interval at a designated distance for assigned rest

**Example:**  
`400 freestyle broken at 100 w/:10R (L4)`

**Execution:**
1. Swim 100, stop
2. Rest 10 seconds
3. Swim 100, stop
4. Rest 10 seconds
5. Swim 100, stop
6. Rest 10 seconds
7. Swim 100, finish
**Total = 400** (counts as ONE interval, but with internal rest breaks)

**Purpose:** Simulate race distance with short recovery to maintain speed

---

#### Breathe Every
**Translation:** During freestyle, take a breath every Xth stroke

**Example:**  
`300 freestyle breathe every 3rd stroke (L3)`

**Execution:** Bilateral breathing (right-left-right or left-right-left pattern)

**Variations:**
- Breathe every 5th = harder (less air)
- Breathe every 7th = very hard
- Breathe every 2nd = easier (good for beginners)

---

### Multi-Set Structures

#### 2× w/:20R:
**Translation:** Do the following mini-set twice, with rest after each complete mini-set

**Example:**
```
2× w/:20R:
  100 freestyle (L5)
  100 your choice stroke or kick (L2)
  1:00R between each set
```

**Execution:**
1. Swim 100 freestyle (L5)
2. Swim 100 choice (L2)
3. Rest 1:00
4. **Repeat entire sequence**
5. Rest :20R after second round

**Total = 400** (two rounds of 200)

---

#### 3×:
**Translation:** Do the following block of swims 3 times

**Example:**
```
3×:
  4×25 IM order by 25 (L3) w/:10R
  4×25 freestyle (L5) w/:15R
  4×25 pull (L3) w/:15R
  150 kick (L2)
```

**Execution:**
- Complete the entire block (4×25 IM, 4×25 free, 4×25 pull, 150 kick)
- Repeat block 2 more times
**Total per round = 450** → **Total = 1350** (3 rounds)

## Standard Workout Structure

Schneider's 100 workouts follow a **4-phase structure:**

### Phase 1: Warm-Up (10-15% of total distance)
**Purpose:**
- Raise heart rate gradually
- Prepare muscles/joints
- Activate nervous system
- Mental preparation

**Typical content:**
- 200-400 choice of stroke (L2)
- 100-200 kick (L2)
- Light drill work

**Example:**
```
200 choice of stroke (L2)
100 kick (L2)
```

---

### Phase 2: Drill / Technique Work (15-20% of total distance)
**Purpose:**
- Refine stroke mechanics
- Practice drills
- Build neuromuscular patterns

**Typical content:**
- 4-8×50 drill/swim combinations
- Isolated drill sets
- Stroke count focus

**Example:**
```
4×50; 25 catch up drill/25 freestyle w/:15R
100 finger tip drag drill
2×100; 50 6-3-6 drill/50 freestyle w/:10R
```

---

### Phase 3: Main Set (60-70% of total distance)
**Purpose:**
- Primary training stimulus
- Highest intensity work
- Build fitness/speed/endurance

**Typical content:**
- Interval sets at L3-L5
- Race-pace work
- Threshold training
- Sprint work

**Example:**
```
5×200 freestyle (L4) w/:30R
300 pull (L3)
8×25 freestyle (L5) w/:15R
```

---

### Phase 4: Cool-Down (5-10% of total distance)
**Purpose:**
- Lower heart rate
- Clear lactate
- Mental transition
- Injury prevention

**Typical content:**
- 100-300 easy swimming (L1-L2)
- Choice of stroke
- Light kicking

**Example:**
```
200 pull (L2)
100 choice of stroke (L1)
```

## Sample Workout Breakdown

### Workout: 1200 Yards/Meters

```
200 choice of stroke (L2)                    [WARM-UP]
100 kick (L2)

100 catch up drill                           [DRILL WORK]

100 pull (L3)                                [PRE-MAIN]
100, 150, 100, 50 freestyle (L4) w/:20R      [MAIN SET]
4×25 freestyle (L5) w/:20R

100 freestyle (L2)                           [COOL-DOWN]

TOTAL: 1200
```

**Distance breakdown:**
- Warm-up: 300 (25%)
- Drill: 100 (8%)
- Main set: 600 (50%)
- Cool-down: 100 (8%)
- Pre-main transition: 100 (8%)

## Workout Progression System

Schneider organizes workouts into **4 tiers** based on total distance:

### Tier 1: 1000-2000 Yards/Meters
**Target audience:** Beginners, fitness swimmers, time-crunched athletes  
**Duration:** 20-35 minutes  
**Workouts in book:** 23 workouts

**Sample:**
```
200 choice (L2)
100 kick (L2)
4×50; 25 drill/25 free w/:15R
200 pull (L3)
4×50 freestyle (L4) w/:20R
100 choice (L2)

TOTAL: 1000
```

---

### Tier 2: 2000-3000 Yards/Meters
**Target audience:** Intermediate swimmers, masters, triathletes  
**Duration:** 35-50 minutes  
**Workouts in book:** 25 workouts

**Sample:**
```
300 choice (L2)
200 kick (L2)
4×50 drill w/:10R
2×200 freestyle (L4) w/:30R
300 pull (L3)
8×25 freestyle (L5) w/:15R
150 choice (L2)

TOTAL: 2500
```

---

### Tier 3: 3000-4000 Yards/Meters
**Target audience:** Competitive age-group, high school, college  
**Duration:** 50-70 minutes  
**Workouts in book:** 34 workouts

**Sample:**
```
300 choice (L2)
200 kick (L2)
6×50 drill w/:10R
5×200 freestyle (L4) w/:45R
300 pull (L3)
10×50 freestyle (L5) w/:30R
200 kick (L2)
200 choice (L2)

TOTAL: 3500
```

---

### Tier 4: 4000-5000 Yards/Meters
**Target audience:** Elite high school, college, post-grad  
**Duration:** 70-90 minutes  
**Workouts in book:** 18 workouts

**Sample:**
```
400 choice (L2)
300 kick (L2)
8×50 drill w/:10R
3×500 freestyle (L4) w/1:00R
400 pull (L3)
12×50 freestyle (L5) w/:30R
200 kick (L2)
300 choice (L2)

TOTAL: 4500
```

## Progressive Overload: Week-by-Week Example

### Week 1: Establish Baseline (1000m)
```
200 choice (L2)
100 kick (L2)
4×50 drill w/:15R
200 freestyle (L3)
100 pull (L2)
4×25 freestyle (L4) w/:20R
100 choice (L2)

TOTAL: 1000
```

---

### Week 2: Add Volume (1200m)
```
200 choice (L2)
100 kick (L2)
4×50 drill w/:15R
300 freestyle (L3)
100 pull (L3)
6×25 freestyle (L4) w/:20R
100 choice (L2)

TOTAL: 1200
```
**Change:** +200m (main set longer)

---

### Week 3: Add Intensity (1400m)
```
200 choice (L2)
100 kick (L2)

200 pull (L3)
4×50 freestyle (L5) w/:30R — now at L5!
100 choice (L2)

TOTAL: 1400
```
**Change:** +200m, higher training levels in main set

---

### Week 4: Consolidate (1400m)
```
Same as Week 3, focus on technique & holding times
```

---

### Week 5: Next Progression (1600m)
```
300 choice (L2)
100 kick (L2)

3×200 freestyle (L4) w/:30R
200 pull (L3)
6×50 freestyle (L5) w/:30R
```

## How to Scale Workouts

### Method 1: Proportional Reduction
**Goal:** Reduce total distance but keep structure

**Original (3000m):**
```
300 choice (L2)
200 kick (L2)
6×50 drill w/:10R
5×200 freestyle (L4) w/:45R
300 pull (L3)
8×25 freestyle (L5) w/:15R
200 choice (L2)

TOTAL: 3000
```

**Scaled to 2000m (66% of original):**
```
200 choice (L2) — reduced from 300
100 kick (L2) — reduced from 200
4×50 drill w/:10R — reduced from 6×50
3×200 freestyle (L4) w/:45R — reduced from 5×200
200 pull (L3) — reduced from 300
6×25 freestyle (L5) w/:15R — reduced from 8×25
100 choice (L2) — reduced from 200

TOTAL: 2000
```

**Key:** Maintain training levels and rest intervals — just reduce reps/distance.

---

### Method 2: Cut a Section
**Goal:** Remove one phase entirely

**Original (2500m):**
```
WARM-UP: 400
DRILL: 300
MAIN SET: 1500
COOL-DOWN: 300
```

**Scaled to 1800m:**
```
WARM-UP: 300 — reduced
DRILL: 200 — reduced
MAIN SET: 1200 — keep most of main set
COOL-DOWN: 100 — minimal
```

**Key:** Protect the main set — that's where adaptation happens.

## Common Mistakes in Workout Execution

### ❌ Mistake 1: Ignoring Rest Intervals
**Problem:** Taking 30 seconds when workout says :15R  
**Why it matters:** Changes intensity, defeats purpose of interval training  
**Fix:** Use pace clock, stick to prescribed rest

---

### ❌ Mistake 2: Wrong Training Level
**Problem:** Swimming L5 when workout calls for L3  
**Why it matters:** Overtraining, can't complete set, ruins workout intent  
**Fix:** Learn what each level feels like (use PE scale)

---

### ❌ Mistake 3: Skipping Warm-Up/Cool-Down
**Problem:** Jumping straight into main set or leaving after last hard swim  
**Why it matters:** Injury risk, poor recovery, mental fatigue  
**Fix:** Treat warm-up/cool-down as non-negotiable

---

### ❌ Mistake 4: Misreading Notation
**Problem:** Doing 4×50 as "swim 200 continuously" instead of "4 separate 50s with rest"  
**Why it matters:** Completely different training stimulus  
**Fix:** Learn notation system, ask coach if unclear

---

### ❌ Mistake 5: Not Tracking Times
**Problem:** No idea if you're getting faster or slower across reps  
**Why it matters:** Can't assess pacing, effort, or improvement  
**Fix:** Write down times for main set, track week-to-week

## Actionable Coaching Applications

### For New Coaches: Use Schneider's Template

**Structure every workout as:**
1. **Warm-up** (15% of total)
2. **Drill work** (15% of total)
3. **Main set** (60% of total)
4. **Cool-down** (10% of total)

**Start here:**
```
[Distance] choice (L2)
[Distance] kick (L2)
[Drill set]
[Main set with clear training levels]
[Distance] choice (L2)

TOTAL: [Calculate]
```

---

### For Intermediate Coaches: Add Variety

**Once athletes know the structure, vary:**
- **Main set format:** Straight sets, pyramids, descends, locomotion
- **Training levels:** Mix L3/L4/L5 within one set
- **Stroke variety:** IM, stroke-specific work

**Example:**
```
300 choice (L2)
200 kick (L2)

4×50 drill your choice w/:10R

8×100 freestyle w/:20R
  Odds: L3
  Evens: L5

300 pull locomotion

200 choice (L2)

TOTAL: 2500
```

---

### For Advanced Coaches: Periodize Workouts

**Week 1 (Base):** High volume, L2-L3  
**Week 2 (Build):** Moderate volume, L3-L4  
**Week 3 (Intensity):** Lower volume, L4-L5  
**Week 4 (Recovery):** Low volume, L1-L2

**Rotate through cycles, track performance.**
